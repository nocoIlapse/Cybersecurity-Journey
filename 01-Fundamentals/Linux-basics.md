# 🐧Linux CLI Cheatsheets

My personal notes on work with the Linux terminal, which I`m keeping while working through Stage 1 on TryHackMe

## 📁Navigation and Working with files
* `pwd` - print currently directory.
* `ls` - view the list of files in the current folder.
* `ls -la` - show all files(inlucing hidden) with detalied information.
* `cd [file_name]` - go to the another folder.
* `cd ..` - go back one folder.
